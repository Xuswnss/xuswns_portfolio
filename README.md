# xuswns Portfolio

카드뉴스 형식 포트폴리오 웹사이트 (Vercel + GitHub Actions 배포)

## 시작하기

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인

## 배포

1. **Vercel**: [vercel.com](https://vercel.com)에서 GitHub 저장소 연동 후 자동 배포
2. **GitHub Actions**: main 브랜치 push 시 빌드 검증 자동 실행

## 문서

`md/` 폴더에 프로젝트 설계 및 설정 설명이 있습니다.

- `1.프로젝트-개요.md`
- `2.폴더구조-설명.md`
- `3.버셀-선택이유-및-설정.md`
- `4.깃허브액션-선택이유-및-설정.md`
- `5.카드뉴스-형식-설명.md`
